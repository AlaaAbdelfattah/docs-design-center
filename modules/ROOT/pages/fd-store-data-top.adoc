Variables are used to store values for use within a flow of a Mule app. After creating a variable, you can access and use it within the scope of a Mule flow where you created it, which includes any flows that are joined with it through a Flow Ref component.


Store and Reuse Data from the Output of a Card:: You can use a Set Variable card to create a variable that stores data from the output of a card. You can then reuse that data in a later card by using the variable.
